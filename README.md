# ⚙️  기능 
- 가상 키보드 
- Form 유효성 검사
- Layout 수정가능 (CSS)
- Form AutoFocusing
- 카드 번호에 따라 변경되는 Card UI

## :zap: 사용방법

```shell
$ npm i react-payment-junyoungapp
$ yarn add react-payment-junyoungapp
```
~~~ jsx
import { PaymentApp } from 'react-payment-junyoungapp';
function App(){
    return (
        <div>
            <PaymentApp/>
        </div>
    )
}

~~~
